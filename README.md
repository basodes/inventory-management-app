# Inventory Management System

A production-ready Inventory Management System built using:

- Spring Boot
- React
- PostgreSQL
- Docker
- Kubernetes
- Jenkins
- Terraform
- Helm
- ArgoCD
- Prometheus
- Grafana
- AWS

## Features

- Product Management
- Category Management
- Supplier Management
- Inventory Tracking
- Authentication
- Reports

Frontend (React)

Backend (Spring Boot)
